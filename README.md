# Monte Carlo

An interactive Monte Carlo simulation engine built with Next.js.

This project will be a browser-based tool for running Monte Carlo simulations, visualizing probability distributions, and stress-testing financial models through randomized sampling. Think of it as a playground for understanding how randomness shapes outcomes, whether you're pricing options, estimating project timelines, or just curious about convergence.

## Planned

- Configurable simulation parameters (iterations, distributions, seed values)
- Real-time visualization of convergence as sample size increases
- Multiple distribution types (normal, log-normal, uniform, custom)
- Portfolio risk analysis through simulated return paths
- Export simulation results

## Tech

Next.js 16, React 19, TypeScript, Tailwind CSS 4

## Status

Scaffolded and ready for implementation.
